Image Comparer Demonstration

Compare two images using analytic geometry.

Open debug console for more information.

Input data are in the form Array of the Object(d, x1, y1, x2, y2). Example:

[
{"d": true, "x1": 10, "y1": 10, "x2": 20, "y2": 20},
{"d": true, "x1": 50, "y1": 60, "x2": 50, "y2": 50},
{"d": true, "x1": 20, "y1": 20, "x2": 30, "y2": 10}
]

You can disable any you can easily disable the corresponding segment by entering the value false in the field d. Example:

[
{"d": true, "x1": 10, "y1": 10, "x2": 20, "y2": 20},
{"d": false, "x1": 50, "y1": 60, "x2": 50, "y2": 50},
{"d": true, "x1": 20, "y1": 20, "x2": 30, "y2": 10}
]
